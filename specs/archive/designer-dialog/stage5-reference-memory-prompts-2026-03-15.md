## Stage 5 Reference And Memory Prompt Handling

Date: 2026-03-15
Stage: Stage 5 - Reference and Memory Prompt Handling

Purpose:
- verify that the designer can turn memory-driven, emotional, and inspiration-style prompts into usable design direction
- ensure those prompts become concrete design framing instead of vague poetic filler

Prompt set:

1. Nostalgic memory prompt
- `I want this to feel like Christmas Eve when I was a kid: warm, quiet, and full of anticipation.`

Expected:
- brief should explicitly capture nostalgia, warmth, quiet anticipation
- proposal should stay concrete and reviewable

2. Inspiration-style prompt
- `I want this to feel like a classic department store holiday window: elegant, glowing, and a little theatrical.`

Expected:
- brief should translate the reference into usable design cues
- proposal should reflect elegance/glow/theatricality without devolving into effect names

3. Emotional indirect prompt
- `The bridge should feel like everything pauses for a breath before the final chorus opens up.`

Expected:
- brief and proposal should translate the emotional phrasing into hold/release design language
- no unnecessary clarification

Pass criteria:
- responses remain concrete enough to produce a strong brief and proposal
- memory/reference phrasing influences the design framing materially
- outputs do not become vague, poetic filler
